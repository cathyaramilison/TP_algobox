# TP\_algobox



\## Objectif

Ce projet a pour objectif de découvrir l’utilisation de GitHub pour le travail collaboratif à partir de programmes réalisés avec le logiciel AlgoBox.



\## Contenu du dépôt

\- Fichiers AlgoBox (.alg) contenant des algorithmes simples : algo1.alg, algo2.alg, algo3.alg

\- Ce fichier read.me



\## Organisation du travail

Le travail a été réalisé en trinôme. Chaque membre modifie un des fichiers AlgoBox puis partage ses changements via les commandes git add, git commit et git push.



\## Outils utilisés

\- AlgoBox

\- Git

\- GitHub



\## Consignes pour executer les fichiers Algobox

-Ouvrir depuis algobox

-Appuyer sur "Tester l'algorithme" (en bas à droite)

-Une fenêtre s'affiche, lancer avec le bouton "Lancer l'algorithme"

-Suivre les instructions sur le terminal de commande
